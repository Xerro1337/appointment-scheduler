# Appointment Scheduler

## Описание проекта

Appointment Scheduler — это веб-приложение для онлайн-записи на приём. Пользователь может выбрать дату и доступный временной слот, указать свои контактные данные и создать запись. Администратор может авторизоваться, просматривать все записи, изменять их статус и удалять их.

## Возможности

- Запись на приём через многошаговую форму
- Выбор доступной даты и времени
- Автоматическая проверка занятости временных слотов
- Проверка корректности введённых данных
- Авторизация администратора с использованием JWT
- Просмотр списка всех записей
- Подтверждение и отмена записи
- Удаление записи
- Уведомления об успешных действиях
- Адаптивный пользовательский интерфейс

## Как сделать локальный запуск

1. Клонировать репозиторий
2. Установить зависимости (`npm install` в папке `server` и `client`)
3. Создать файл `.env` в папке `server`
```
DATABASE_URL=...
ADMIN_LOGIN=...
ADMIN_PASSWORD=...
JWT_SECRET=...
```
4. Запустить (`npm run dev` в `client` и `npm start` в `server`)
